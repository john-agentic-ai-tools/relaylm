# Configuration Reference

RelayLM configuration is stored in `~/.config/relaylm/config.yml`. This file is auto-generated by `relaylm setup` and modified by `relaylm providers add`.

## File Location

```
~/.config/relaylm/config.yml
```

## Schema

### Top-Level Fields

```yaml
version: 1
container_runtime: podman
models:
  - name: Qwen/Qwen3-0.6B
    source: huggingface
    gpu_index: null
    args: {}
fallback:
  order:
    - local
    - anthropic
  timeout_seconds: 30
router:
  host: 127.0.0.1
  port: 8000
providers:
  anthropic:
    enabled: true
    base_url: https://api.anthropic.com/v1
    keychain_service: relaylm-anthropic
```

### Field Descriptions

| Field | Type | Required | Description |
|-------|------|----------|-------------|
| `version` | integer | yes | Config schema version |
| `container_runtime` | string | no | Runtime to use: `podman` or `docker` |
| `models` | array | yes | List of deployed models |
| `models[].name` | string | yes | Hugging Face model ID |
| `models[].source` | string | yes | Model source (e.g., `huggingface`) |
| `models[].gpu_index` | int or null | no | GPU device index |
| `models[].args` | object | no | Additional vLLM arguments |
| `fallback.order` | array | no | Provider fallback priority |
| `fallback.timeout_seconds` | int | no | Fallback request timeout |
| `router.host` | string | no | Router bind address |
| `router.port` | int | no | Router port |
| `providers` | object | no | Configured external providers |
| `providers.{name}.enabled` | bool | yes | Whether provider is active |
| `providers.{name}.base_url` | string | yes | Provider API base URL |
| `providers.{name}.keychain_service` | string | yes | Keychain service name for API key |

## Example: Full Configuration with Multiple Providers

```yaml
version: 1
container_runtime: podman
models:
  - name: Qwen/Qwen3-4B
    source: huggingface
    gpu_index: 0
    args:
      max_model_len: 4096
fallback:
  order:
    - local
    - anthropic
    - openai
  timeout_seconds: 60
router:
  host: 0.0.0.0
  port: 8000
providers:
  anthropic:
    enabled: true
    base_url: https://api.anthropic.com/v1
    keychain_service: relaylm-anthropic
  openai:
    enabled: true
    base_url: https://api.openai.com/v1
    keychain_service: relaylm-openai
```
