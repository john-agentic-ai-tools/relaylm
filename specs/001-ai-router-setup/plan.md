## Summary

Build a single-command CLI (`relaylm setup`) that bootstraps a complete local
AI routing environment. It auto-detects hardware (RAM, CPU, GPU), selects
matching models from Hugging Face, sets up vLLM inside Podman/Docker,
configures external providers (Anthropic, OpenAI) with keychain-backed secret
storage, and auto-configures coding agents (Claude Code, OpenCode) to point
at the local router. Configuration is re-runnable with timestamped backups
for safe rollback.

## Technical Context

**Language/Version**: Python 3.11+
**Primary Dependencies**: typer (CLI), vLLM (Docker image), huggingface-hub,
  keyring (system keychain), Podman/Docker CLI wrappers, pyyaml
**Storage**: File-based config (`~/.config/relaylm/config.yml`); system
  keychain for API keys; timestamped backups in `~/.config/relaylm/backups/`
**Testing**: pytest + pytest-cov; testcontainers for container integration tests
**Target Platform**: Linux, macOS (Windows via WSL2)
**Project Type**: CLI tool (pip-installable Python package)
**Performance Goals**: Setup completes in under 10 minutes on 16 GB RAM
  workstation; CLI flag parsing and config writes sub-second
**Constraints**: Minimum 8 GB free RAM for smallest model; internet required
  on first run for image/model pulls; sudo access for container runtime
**Scale/Scope**: Single-machine deployment for one developer
